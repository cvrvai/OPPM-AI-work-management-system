# Microservice Map

## Current runtime surfaces

- `frontend/`: user-facing application surface
- `gateway/`: gateway and API edge surface
- `services/`: domain and backend services
- `shared/`: shared contracts and utilities
- `deploy/`, `monitoring/`, and `scripts/`: operational support surfaces
